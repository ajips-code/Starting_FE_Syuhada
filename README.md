## Syuhada FE Starting Move
1. Remove package-lock.json and then Install all depedencies
   ### npm install
   atau
   ### yarn install
2. Install vite depedencies if haven't installed
   ### npm install vite --save-dev
   or
   ### yarn add vite --dev
3. To see dev web
   ### npm run dev
4. To see preview
   ### npm run preview
5. To build all dev
   $$# npm run build
